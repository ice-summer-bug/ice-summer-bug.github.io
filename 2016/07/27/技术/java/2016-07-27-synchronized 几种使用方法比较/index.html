synchronized 几种使用方法比较

synchronized 关键字在 Java 中出现的很早， Java 提供的实现原子性的内置锁机制，就是用这个关键字实现。
synchronized 关键字可以修饰方法，也可以修饰一个代码块，实现一种互斥锁，这里就不做赘述。
